# 🎓 Troubleshooters Academy Website

A modern and responsive academy website developed to provide students with complete information about courses, facilities, teachers, and contact details in an interactive and user-friendly interface.

## 🚀 Project Overview

The Troubleshooters Academy Website is designed to help educational institutes establish a strong online presence. The website includes multiple pages with responsive layouts, smooth navigation, dynamic sections, and modern UI/UX design.

This project was developed using frontend and backend web technologies and deployed locally using XAMPP.

## ✨ Features

* Responsive Home Page
* Courses Section
* Teachers/Faculty Section
* Facilities Section
* Contact Page
* Smooth Navigation Bar
* Interactive Teacher Cards Slider
* Dynamic PHP Integration
* MySQL Database Connectivity
* Clean and Modern UI
